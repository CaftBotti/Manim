Manim FourierOfPaths Examples

This example is for users to make Fourier animations quickly. Enjoy the beauty of maths!

For more information, See: https://github.com/CaftBotti/Manim/blob/main/ManimFourierOfPathsExamples

#!/usr/bin/env python

# This code uses the Fourier base: https://github.com/CaftBotti/Manim/blob/main/ManimFourier
# This code shows number 0~9, A~Z and a~z.
# You can ignore zoomed camera if you are using FourierOfPaths because it has no effects. But in other modes it's ok.
# FourierFromSVG mode and FourierTexSymbol mode cannot show shapes or characters contain "annulus", like 0 and 8,9,R correctly.

class One(FourierOfPaths):
    CONFIG = {
        'n_vectors': 150,
        'run_time': 100,
        'slow_factor': 0.2,
        'tex_class': TextMobject,
        'tex': '1',
        'include_zoom_camera': True,
        'zoom_position': lambda zc: zc.to_corner(DR),
        'zoom_factor': 0.1,
        'wait_before_start': 3
    }
